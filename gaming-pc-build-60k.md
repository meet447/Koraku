## 🛠️ Build Tips

1. **Enable XMP in BIOS** — This unlocks the full 3200MHz RAM speed (default is usually 2133MHz)
2. **Install GPU drivers first** — Download from NVIDIA/AMD website, not Windows Update
3. **Use two RAM sticks** — Dual-channel mode gives ~15% better performance than single stick
4. **Cable management** — Good airflow = lower temps = better performance
5. **Update BIOS** — Latest BIOS ensures best compatibility with Ryzen 5000
